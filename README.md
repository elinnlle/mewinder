# Mewinder (Кототиндер) 🐱

Учебное приложение на Flutter: развлекательное приложение, позволяющее свайпать случайных котиков, смотреть их породы и изучать детальную информацию.

## Реализованные фичи

- [x] Главный экран со случайным котиком
- [x] Загрузка котиков из TheCatAPI
- [x] Отображение названия породы
- [x] Лайк/дизлайк и свайпы
- [x] Детальный экран котика
- [x] Таббар и экран списка пород
- [x] Детальный экран породы
- [x] Обработка сетевых ошибок
- [x] Кастомная иконка приложения
- [x] Сохранение лайкнутых котиков  
- [x] Плавные анимации свайпов  
- [x] Анимированный фон  
- [x] Анимация лайков возле счётчика
- [x] Предзагрузка следующего котика для ускорения загрузки
- [x] Лоадер-карточки с Shimmer

## Архитектура проекта

Проект построен с использованием **Clean Architecture**.

Структура приложения разделена на слои:

- **Presentation** — UI, страницы и контроллеры состояния
- **Domain** — бизнес-логика и use-cases
- **Data** — работа с API и локальным хранилищем

Зависимости между слоями направлены только внутрь приложения (UI → Domain → Data).

Для управления зависимостями используется **GetIt (Dependency Injection)**.

## Работа с API-ключом

Для работы с TheCatAPI используется API-ключ.  
Он **не хранится в репозитории** и добавляется только локально (файл с ключом указан в `.gitignore`, чтобы избежать его публикации).

### Где хранится ключ локально
```
lib/secrets.dart
```

Пример содержимого:
```
const catApiKey = "YOUR_API_KEY";
```

Каждому разработчику необходимо создать этот файл у себя вручную и вставить собственный ключ.
